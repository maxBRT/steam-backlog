# Steam Backlog (Laravel)

Laravel + Inertia.js + React application. Stack decisions: [ADR-0001](../docs/adr/0001-laravel-inertia-stack.md).

## Local setup

```bash
composer install
cp .env.example .env   # if needed
php artisan key:generate
touch database/database.sqlite
php artisan migrate
npm install
npm run build
```

### Required secrets

After copying `.env.example`, you need to configure secrets:

1. **Steam API key** (required for library sync):
   - Get one at [steamcommunity.com/dev/apikey](https://steamcommunity.com/dev/apikey)
   - Add to `.env` as `STEAM_API_KEY=your_key_here`

2. **Application key** (auto-generated by `php artisan key:generate`):
   - Used for encryption and sessions
   - Never commit this to version control

See [docs/secrets-management.md](../docs/secrets-management.md) for complete secrets documentation.

## Development

```bash
composer dev
```

Runs the PHP server, queue worker, log tail, and Vite dev server together.

Or separately:

```bash
php artisan serve
php artisan queue:work
npm run dev
```

## Production database

Use PostgreSQL on Laravel Cloud. Set `DB_CONNECTION=pgsql` and the corresponding `DB_*` variables in the deployment environment.
